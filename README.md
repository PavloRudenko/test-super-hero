## Installation server app

```bash
# install server dependencies
$ cd server

$ npm install
```

## Running the server app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Installation client app

```bash
# install client dependencies
$ cd client

$ npm install
```

## Running the client app

```bash
# development
$ npm run start

# production mode
$ npm run build
```
